Comandos utiles:

git status - muestra el estado actual del proyecto, si hay cambios pendientes.

git add - a√±adir los cambios al staging

git commit -m "comentario" - confirmar los cambios que estan en staging

git log - muestra un historial de todos los commits

git show - muestra info de los commits

git branch nombre-rama - crea una nueva rama

git branch - muestra todas las ramas existentes, y en cual se encuentra el HEAD

git merge nombre-rama - fusiona la rama especificada con la actual

git checkout - Cambiarse a otra rama

git restore - volver a un commit anterior

git pull origen-rama - trae y fusiona del origen remoto la rama especificada con la rama actual

git push origen-rama - envia y fusiona al origen remoto la rama actual con la especificada

gitk - Abre un programa que muestra toda la historia de los commits

Otros comandos: git rebase (parecido a un merge), git stash (guardar cambios en memoria) git clean (borrar archivos), git cherry-pick (traer un commit viejo al head de una rama)
